## 1. 배경 (Introduction)

사람의 걸음걸이는 놀라울 정도로 복잡한 운동이다. 200개 이상의 뼈와 600개 이상의 근육이 신경계의 지휘 아래 정밀하게 협응하여 한 걸음이 만들어진다. 건강한 사람은 이 과정을 의식하지 않지만, 뇌졸중·뇌성마비·파킨슨병 등의 질환이 이 시스템의 어느 한 부분을 손상시키면 **병리적 보행(pathological gait)** — 비정상적인 걸음걸이가 나타난다.

문제는 '어디가 얼마나 잘못되었는가'를 정확히 판단하는 것이 쉽지 않다는 점이다. 숙련된 임상가의 육안 관찰조차 검사자 간 일치도가 낮고(Cohen's κ ≈ 0.3–0.6), 눈에 보이지 않는 관절 내부의 힘과 모멘트는 파악할 수 없다 [Baker, 2006]. 이러한 한계를 극복하기 위해 1970년대부터 발전해 온 것이 **임상 보행 분석(Clinical Gait Analysis, CGA)** — 카메라·힘판·근전도 등 계측 장비로 걸음걸이를 수치화하고, 이를 치료 의사결정에 활용하는 분야이다 [Baker et al., 2023].

## 2. 기초 개념 (Foundations)

### 보행 주기: 걸음의 해부학 [기초 지식]

보행을 이해하려면 먼저 **보행 주기(gait cycle)**라는 단위를 알아야 한다. 오른발 뒤꿈치가 바닥에 닿는 순간부터 같은 오른발이 다시 바닥에 닿을 때까지가 하나의 주기이다.

이 주기를 비유하자면 **시계의 한 바퀴**와 같다:
- **입각기(stance phase, 0–60%)**: 발이 바닥에 닿아 있는 시간. 시계의 6시에서 12시를 거쳐 다시 4시까지의 긴 구간. 체중을 지탱하고, 몸을 앞으로 밀어내는 "일하는 시간"이다.
- **유각기(swing phase, 60–100%)**: 발이 공중에 떠서 앞으로 나가는 시간. 시계의 4시에서 6시까지의 짧은 구간. 다음 걸음을 준비하는 "이동 시간"이다.

입각기는 다시 4단계로 나뉜다:
1. **하중 반응기(loading response)**: 발이 닿자마자 체중을 받는 충격흡수 구간
2. **중간 입각기(mid-stance)**: 한 다리로 서서 무게중심이 발 위를 넘어가는 구간
3. **말기 입각기(terminal stance)**: 뒤꿈치가 들리며 추진력을 만드는 구간
4. **전유각기(pre-swing)**: 반대쪽 발이 착지하고, 이제 발을 들 준비를 하는 구간

### 보행 분석의 세 가지 눈 [기초 지식]

임상 보행 분석은 걸음걸이를 세 가지 렌즈로 본다:

1. **시공간 매개변수**: "얼마나 빨리, 얼마나 넓게 걷는가" — 보행속도, 보폭, 분속수 등. 자동차의 계기판과 같아서 전반적 상태를 빠르게 파악한다.

2. **운동학(kinematics)**: "관절이 얼마나 움직이는가" — 발목·무릎·고관절·골반의 각도 변화를 시간에 따라 그래프로 나타낸다. 관절의 "움직임 지도"이다.

3. **운동역학(kinetics)**: "관절에 어떤 힘이 작용하는가" — 지면반력, 관절 모멘트, 관절 파워. 겉으로 보이지 않는 내부의 원인을 드러내는 "X-ray"와 같다. 이것이야말로 단순 관찰로는 알 수 없는, 계측 보행 분석의 핵심 가치이다.

## 3. 핵심 개념 (Deep Dive)

### 3.1 병리적 보행의 원인과 분류: "오케스트라 비유"

정상 보행을 **오케스트라 연주**에 비유하면, 뇌(지휘자) → 척수(부지휘자) → 신경(악보 전달자) → 근육(연주자) → 뼈와 관절(악기)이 조화를 이루어야 아름다운 음악(정상 보행)이 나온다.

각 부분이 손상되면 다른 종류의 "불협화음"이 발생한다:

**지휘자가 쓰러진 경우 (뇌졸중):**
뇌의 한쪽이 손상되면 반대쪽 몸의 근육 제어가 어려워진다. 환자는 마치 한쪽 팔다리로만 연주해야 하는 오케스트라처럼, 환측 다리를 반원 모양으로 돌려 내딛는 **환선보행(circumduction gait)**을 보인다. 발끝이 처지는 족하수(foot drop)를 보상하기 위해 골반을 들어올리거나(hip hiking), 다리 전체를 바깥으로 돌려 지면에 걸리지 않게 한다.

**연주자들의 긴장이 풀리지 않는 경우 (뇌성마비):**
뇌성마비에서는 근육이 과도하게 긴장(경직, spasticity)하여 마치 기타 줄이 너무 팽팽한 것처럼 관절이 자유롭게 움직이지 못한다. 경직 정도와 부위에 따라:
- **첨족보행(equinus):** 종아리 근육이 팽팽 → 발꿈치를 딛지 못하고 발가락으로 걸음 (발레리나처럼)
- **구부정 보행(crouch gait):** 무릎·고관절이 펴지지 않아 앉은 듯한 자세로 걸음
- **가위보행(scissors gait):** 양쪽 다리가 안으로 교차되며 걸음

**지휘자가 떨리는 경우 (파킨슨병):**
도파민 부족으로 움직임 시작과 크기 조절이 어려워진다. 걸음이 점점 작아지고(보폭 감소), 한번 시작하면 점점 빨라지는 **가속보행(festination)** — 마치 내리막길에서 굴러가는 것처럼 — 이 나타나며, 갑자기 발이 바닥에 붙은 듯 멈추는 **동결현상(freezing of gait)**이 발생한다.

### 3.2 임상 보행 분석의 작동 원리: "GPS 내비게이션 비유"

임상 보행 분석 시스템을 **자동차 내비게이션**에 비유할 수 있다:

1. **위치 추적 (Motion Capture):** GPS 위성이 자동차 위치를 추적하듯, 8–16대의 적외선 카메라가 환자 몸에 부착된 반사 마커(보통 30–50개)의 3D 위치를 초당 100–200회 추적한다. 이를 통해 각 관절이 매 순간 몇 도 구부러져 있는지(운동학) 계산한다.

2. **도로 상태 감지 (Force Plates):** 도로 위의 센서가 차량 하중을 측정하듯, 바닥에 매립된 힘판(force plate)이 발이 바닥을 밀어내는 힘(지면반력, GRF)을 3차원으로 측정한다. 수직 성분은 체중의 약 1.2배(착지 충격)에서 0.8배(중간 입각기)까지 특징적인 M자 형태를 보인다.

3. **엔진 상태 모니터링 (EMG):** 엔진 센서가 각 실린더의 작동을 모니터링하듯, 피부에 부착한 전극이 근육의 전기 신호(EMG)를 측정한다. 어떤 근육이 보행 주기의 어느 시점에 활성화되는지 확인하여, **타이밍 이상**(정상 시점에 꺼져야 할 근육이 켜져 있는 경우 등)을 진단한다.

4. **경로 분석 (Inverse Dynamics):** 내비게이션이 실시간 교통·지형 정보를 종합하여 최적 경로를 분석하듯, 역동역학(inverse dynamics)은 마커 궤적 + 지면반력 + 인체 계측 데이터를 뉴턴 역학으로 통합하여 **관절 내부의 힘과 모멘트**를 계산한다. 이것이 보행 분석의 가장 강력한 도구이며, 예를 들어 "무릎이 구부러진 것이 햄스트링 경직 때문인가, 대퇴사두근 약화 때문인가"를 구별할 수 있게 한다.

### 3.3 요약 지표: "보행의 건강 점수"

보행 분석 데이터는 방대하다 — 한 환자의 검사에서 수백 개의 시계열 데이터가 나온다. 이를 직관적으로 소통하기 위해 여러 **요약 지표(summary index)**가 개발되었다:

**Gait Deviation Index (GDI):**
15개 운동학 변수를 하나의 숫자로 요약한다. 정상군 = 100점, 10점 감소할 때마다 정상에서 1 표준편차만큼 벗어남을 의미한다. 마치 신용점수처럼 전반적 보행 건강을 한눈에 보여주지만, 구체적으로 어디가 문제인지는 알려주지 않는다.

**Gait Profile Score (GPS):**
GDI와 유사하나, 9개 운동학 변수 각각의 편차(°)를 제공하므로 "무릎 시상면에서 5.2° 벗어남"처럼 구체적 위치를 파악할 수 있다. MCID ≈ 1.6° — 이 이상의 변화가 있어야 임상적으로 의미 있다 [Robinson et al., 2017].

**Edinburgh Visual Gait Score (EVGS):**
비디오만으로 평가 가능한 17항목 관찰 점수(0–34점). 장비가 필요 없어 접근성이 높으나, 검사자 훈련이 필요하다. MCID ≈ 2.4점.

### 3.4 데이터에서 의사결정까지: 실제 임상 워크플로

보행 분석의 궁극적 가치는 **치료 의사결정 개선**에 있다. 전형적인 워크플로:

1. **의뢰:** 정형외과/재활의학과에서 수술 또는 재활 계획 수립을 위해 의뢰
2. **데이터 수집:** 맨발 + 보조기 착용 조건에서 각 5–10회 보행 시행
3. **데이터 처리:** 소프트웨어에서 자동/반자동 처리 후 품질 검사
4. **다학제 회의:** 의사·물리치료사·보행분석가·공학자가 데이터를 함께 검토
5. **보고서:** 주요 소견, 정상 대비 편차, 치료 권고안 포함

Wren et al.(2020)의 체계적 문헌고찰에 따르면, 3DGA 데이터가 치료 계획을 변경시킨 비율은 40–89%이며, RCT 2건에서 3DGA 권고를 따랐을 때 환자 결과가 유의하게 개선되었다.

## 4. 수식 구현 (Key Formulas)

### 4.1 역동역학 (Inverse Dynamics)

세그먼트 $i$의 선형 운동방정식:

$$\vec{F}_{proximal} = m_i \vec{a}_{COM} - \vec{F}_{distal} - m_i \vec{g}$$

각운동방정식 (질량중심 기준):

$$\vec{M}_{proximal} = I_i \vec{\alpha} + \vec{\omega} \times (I_i \vec{\omega}) - \vec{M}_{distal} - \vec{r}_{distal} \times \vec{F}_{distal} - \vec{r}_{proximal} \times \vec{F}_{proximal}$$

관절 파워:

$$P_{joint} = \vec{M}_{joint} \cdot \vec{\omega}_{joint}$$

```python
import numpy as np

def inverse_dynamics_segment(m, I, a_com, alpha, omega,
                              F_distal, M_distal,
                              r_distal, r_proximal, g=np.array([0, -9.81, 0])):
    """
    단일 세그먼트의 역동역학 계산 (Newton-Euler).

    Parameters
    ----------
    m : float           — 세그먼트 질량 [kg]
    I : (3,3) array     — 관성 텐서 [kg·m²]
    a_com : (3,) array  — 질량중심 선가속도 [m/s²]
    alpha : (3,) array  — 각가속도 [rad/s²]
    omega : (3,) array  — 각속도 [rad/s]
    F_distal : (3,) array  — 원위 관절 반력 [N]
    M_distal : (3,) array  — 원위 관절 모멘트 [N·m]
    r_distal : (3,) array  — COM에서 원위 관절까지 벡터 [m]
    r_proximal : (3,) array — COM에서 근위 관절까지 벡터 [m]
    g : (3,) array      — 중력 가속도 벡터 [m/s²]

    Returns
    -------
    F_proximal : (3,) — 근위 관절 반력
    M_proximal : (3,) — 근위 관절 모멘트
    """
    # 선형 운동방정식: ΣF = ma
    F_proximal = m * a_com - F_distal - m * g

    # 각운동방정식: ΣM_COM = Iα + ω × (Iω)
    gyroscopic = np.cross(omega, I @ omega)
    M_proximal = (I @ alpha + gyroscopic
                  - M_distal
                  - np.cross(r_distal, F_distal)
                  - np.cross(r_proximal, F_proximal))

    return F_proximal, M_proximal


def joint_power(M_joint, omega_joint):
    """관절 파워 = 모멘트 · 각속도 (내적)"""
    return np.dot(M_joint, omega_joint)  # [W]
```

### 4.2 Gait Deviation Index (GDI) 계산

$$GDI = 100 - 10 \cdot \ln\left(\frac{d_{patient}}{d_{control,mean}}\right)$$

여기서 $d$는 15개 운동학 변수를 SVD로 차원 축소한 주성분 공간에서의 유클리드 거리이다.

```python
import numpy as np

def compute_gdi(patient_waveforms, control_database):
    """
    Gait Deviation Index 계산.

    Parameters
    ----------
    patient_waveforms : (15, 51) array
        15개 kinematic variables × 51 time points (0–100% GC, 2% 간격)
        변수 순서: pelvic tilt/obliquity/rotation, hip flex/add/rot,
                   knee flex, ankle dorsi, foot progression (좌+우)
    control_database : (N, 15, 51) array
        정상군 N명의 동일 구조 데이터

    Returns
    -------
    gdi : float — 100이 정상, 낮을수록 이상
    """
    N = control_database.shape[0]

    # Flatten to 1D vectors (15 × 51 = 765 차원)
    patient_vec = patient_waveforms.flatten()           # (765,)
    control_vecs = control_database.reshape(N, -1)      # (N, 765)

    # 정상군 평균 계산
    mean_control = control_vecs.mean(axis=0)            # (765,)

    # SVD (정상군 중심화 후)
    centered = control_vecs - mean_control              # (N, 765)
    U, S, Vt = np.linalg.svd(centered, full_matrices=False)

    # 분산 99% 커버하는 주성분 수
    explained_var = np.cumsum(S**2) / np.sum(S**2)
    n_comp = int(np.argmax(explained_var > 0.99)) + 1

    # 주성분 공간에 투영
    basis = Vt[:n_comp]                                 # (n_comp, 765)
    patient_scores = basis @ (patient_vec - mean_control)
    control_scores = (basis @ centered.T).T             # (N, n_comp)

    # 거리 계산 → GDI
    d_patient = np.linalg.norm(patient_scores)
    d_controls = np.mean(np.linalg.norm(control_scores, axis=1))

    gdi = 100 - 10 * np.log(d_patient / d_controls)
    return round(gdi, 1)
```

### 4.3 좌우 비대칭 지수 (Symmetry Index)

$$SI = \frac{|X_{affected} - X_{unaffected}|}{0.5 \times (X_{affected} + X_{unaffected})} \times 100\%$$

```python
def symmetry_index(x_affected, x_unaffected):
    """
    Robinson의 대칭 지수. 0% = 완전 대칭.

    Parameters
    ----------
    x_affected : float — 환측 값 (예: step length)
    x_unaffected : float — 건측 값

    Returns
    -------
    si : float — 비대칭 지수 [%]
    """
    mean_val = 0.5 * (x_affected + x_unaffected)
    if mean_val == 0:
        return 0.0
    return abs(x_affected - x_unaffected) / mean_val * 100
```

|------|------|
| **객관적 정량화** | 숫자로 표현되어 시간 경과에 따른 변화 추적, 치료 전후 비교가 명확 |
| **숨겨진 원인 탐지** | 역동역학으로 근력 약화 vs 경직 vs 보상 패턴을 구별 가능 |
| **치료 계획 개선** | 3DGA 데이터가 치료 계획을 40–89% 변경시킴 [Wren et al., 2020] |
| **다학제 소통 도구** | 그래프·수치로 의사·치료사·공학자 간 공통 언어 제공 |
| **연구 표준화** | 재현 가능한 프로토콜로 다기관 연구, 메타분석 가능 |

### 단점

| 측면 | 내용 |
|------|------|
| **높은 비용** | 실험실 구축 $100K–$500K+, 검사당 $1,000–$3,000 |
| **측정 오차** | 연조직 아티팩트(대퇴부 최대 ~20mm), 횡단면 회전 데이터 신뢰성 낮음 |
| **실험실 환경의 한계** | 10m 보행로에서의 "인위적" 걸음 — 일상 보행과 다를 수 있음 |
| **전문 인력 의존** | 데이터 수집·해석에 고도 훈련 필요, 표준화 프로토콜 부재 |
| **제한된 RCT 근거** | 환자 결과 개선을 입증한 RCT는 2건에 불과 |
| **단일 지표의 함정** | GDI/GPS 같은 요약 점수는 서로 다른 병리가 동일 수치를 산출 가능 |

## 6. 총평 (Conclusion)

임상 보행 분석은 50년간의 발전을 거쳐, 특히 뇌성마비 아동의 수술 계획 수립에서 사실상의 표준(de facto standard)으로 자리 잡았다. 역동역학을 통해 겉으로 보이는 보상 패턴 뒤에 숨은 진짜 원인을 찾아낼 수 있다는 것이 핵심 가치이며, 이는 육안 관찰만으로는 불가능한 영역이다.

그러나 높은 비용과 전문 인력 의존이라는 진입 장벽, 그리고 환자 결과 개선에 대한 높은 수준의 근거(RCT)가 아직 부족하다는 점은 분명한 한계이다. 이러한 gap을 메우기 위해 **웨어러블 센서(IMU)**, **스마트폰 기반 markerless 분석(OpenCap 등)**, **AI 기반 자동 분류** 기술이 빠르게 발전하고 있으며, 이는 보행 분석을 실험실 밖의 일상 환경으로 확장시킬 잠재력을 가진다 [Moissenet et al., 2022].

도입 권고: 뇌성마비 다단계 수술 계획, 뇌졸중 재활 효과 모니터링, 파킨슨병 약물 반응 평가 등 **치료 의사결정의 불확실성이 높은 상황**에서 가장 높은 부가가치를 제공한다. 비용 대비 효과를 고려하면, 모든 환자에게 적용하기보다 **치료 방향이 불분명한 복잡한 사례**에 집중 투입하는 것이 현실적이다.
