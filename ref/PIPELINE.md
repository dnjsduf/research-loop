## --update 모드

```
./ralph.sh "주제" --update

┌─────────────────┐     ┌──────────────────┐     ┌────────────────┐
│ 캐시 삭제       │ ──→ │ research-engine  │ ──→ │ fetch-sources  │
│ research JSON   │     │ 최신 논문 재검색  │     │ PDF 다운로드   │
└─────────────────┘     └──────────────────┘     └───────┬────────┘
                                                          │
                                                          ▼
┌─────────────────────────────────────────────────────────────────┐
│  Claude — UPDATE 전용 프롬프트 (PDF 최대 5개 첨부)               │
│  1. 기존 knowledge + report 읽기                                 │
│  2. 새 research JSON과 비교 → 미반영 논문 파악                   │
│  3. 웹 검색으로 최신 동향 추가 조사                               │
│  4. update.md 병합 정책 적용 (충돌 시 버전 분기)                  │
│  5. verify-knowledge + verify-report                             │
│  6. Changelog 기록                                               │
│  7. queue에서 source_of 하위 pending → 일괄 done                 │
└──────────────────────────────┬──────────────────────────────────┘
                               │
                               ▼
                    git commit + push
```

## 파일 생성 맵

```
프로젝트/
├── [파이프라인 코드]
│   ├── ralph.sh              ← 오케스트레이터
│   ├── research-engine.sh    ← 학술 탐색
│   ├── fetch-sources.sh      ← PDF 다운로드
│   ├── detect-python.sh      ← Python 감지
│   ├── queue-util.py         ← queue 파서
│   ├── PROMPT.md             ← Claude 마스터 프롬프트
│   ├── add-knowledge.md      ← 문서 생성 지침
│   ├── update.md             ← 병합 정책
│   ├── verify-knowledge.md   ← 지식DB 검증
│   └── verify-report.md      ← 보고서 검증
│
├── [자동 생성 — 상태 관리]
│   ├── queue.md              ← 처리 대기열
│   ├── activity.md           ← 실행 로그
│   └── inaccessible-papers.txt ← 접근 불가 논문
│
└── docs/
    ├── research/             ← research-engine 결과
    │   └── {slug}.json          (캐시 24h)
    ├── sources/              ← 원본 PDF (git 제외)
    │   └── {slug}.pdf
    ├── knowledge/            ← AI 지식 DB
    │   └── {slug}.md
    └── reports/              ← 사람용 보고서
        └── {slug}_report.md
```
